# Security Policy

Thank you for taking an interest in the security of this project.

This document explains how to report security vulnerabilities related to this repository.

---

## Supported Versions

Security updates are only provided for the latest version of this project.

| Version | Supported |
| ------- | --------- |
| Latest | ✅ |
| Older Versions | ❌ |

---

## Reporting a Vulnerability

If you discover a possible security vulnerability, please report it privately.

Please include the following information:

- A clear description of the vulnerability.
- Steps to reproduce the issue.
- The potential impact.
- Any additional information that may help with investigation.

## Response Process

After receiving a security report, reasonable efforts will be made to:

- Review and investigate the reported issue.
- Confirm whether the issue is a valid security vulnerability.
- Provide updates when appropriate.
- Fix confirmed security issues where possible.

---

## Responsible Disclosure

Please do not publicly disclose any security vulnerability before it has been reviewed and, if necessary, resolved.
